Eclipse: change jdk 7 instead jre. Eclipse-> Preferences -> Java -> Installed Jres -> Add and search the folder where you installed JDK 7.

Tomcat: a la hora de crear el certificado buscar la carpeta Java -> jdk... -> bin y ejecutar el comando keytool -keygen -alias tomcat -keyalg RSA , a continuación, 
introducir la clave changeit y no cambiar nada mas. Usar la misma clave para el certificado y el almacen.
https://tomcat.apache.org/tomcat-7.0-doc/ssl-howto.html
Archivo que hay que descomentar y cambiar el puerto esta en --> C:\xampp\tomcat\conf (no olvidar cambiar tambien redirectPort de la conexion sin SSL )
<Connector port="8181" protocol="org.apache.coyote.http11.Http11Protocol"
               maxThreads="150" SSLEnabled="true" scheme="https" secure="true"
			   keystoreFile="${user.home}/.keystore" keystorePass="changeit"
               clientAuth="false" sslProtocol="TLS" />
